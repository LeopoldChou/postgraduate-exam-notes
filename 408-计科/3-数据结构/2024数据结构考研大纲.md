# 2024 数据结构考研大纲（占 45 分）



### 考查目标

1、掌握数据结构的基本概念、基本原理和基本方法。

2、掌握数据的逻辑结构、储存结构及基本操作的实现，能够对算法进行基本的时间复杂度与空间复杂度的分析。

3、能够运用数据结构基本原理和方法进行问题的分析与求解，具备采用 C 或 C++ 语言设计与实现算法的能力。
